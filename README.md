# CookBuddy
HCI (CS 330) Team Project - Anant, Anton, Ethan, Vivian

Hosted URL: [cookbuddy](https://cookbuddy-4abd2.web.app/)

## Setup
1. Clone the repo
2. Run `npm install`
3. Add a `.env` file (ask a teammate for the Firebase config and OpenAI API key)
4. Run `npm run dev`

## Redeployment After Changes
1. Run `npm run build`
2. Run `firebase deploy`
